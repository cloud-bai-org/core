# 變更提案：offering-arrangement

## 為什麼需要這個變更

供品擺設是拜拜儀式中不可或缺的環節。目前系統尚未提供任何供品管理功能，使用者無法在線上進行虛擬供品擺設，也無法記錄實體供品。此變更旨在補足儀式流程中「供品準備與擺設」的核心缺口，讓線上與線下拜拜體驗更加完整。

## 變更內容

### 新增功能

1. **線上虛擬供品選擇（拜神明模式）** - 提供虛擬供品庫，使用者可瀏覽分類並選擇供品加入儀式
2. **虛擬供品拖放擺設（拜神明模式）** - 支援拖放方式將供品擺設於虛擬供桌上，可調整位置與移除
3. **實體供品紀錄（兩種模式皆適用）** - 支援拍照記錄實體供品，並可掃描供桌 QR Code 關聯儀式
4. **實體供品與計時連動（兩種模式皆適用）** - 焚香計時結束時推播通知使用者領取供品

## 能力範圍 (Capabilities)

- `offering-arrangement`

## 影響評估

### 影響範圍

- **儀式流程**：在儀式步驟中新增供品擺設環節
- **拜神明模式**：新增虛擬供品庫、供桌拖放互動介面
- **兩種模式共用**：新增實體供品拍照、QR Code 掃描、計時連動推播
- **後端服務**：需新增供品資料模型、圖片上傳、QR Code 辨識、推播通知整合
