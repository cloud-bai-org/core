# 提案：環保影響追蹤器

## 概述

為雲端拜拜平台新增環保影響追蹤功能，讓使用者在每次線上儀式完成後，能具體看到相較於傳統實體拜拜所減少的環境影響，並透過累積統計、視覺化圖表與里程碑機制，持續激勵使用者採用線上拜拜。

## 動機

傳統拜拜儀式涉及大量金紙燃燒與紙紮焚化，產生可觀的碳排放與紙張消耗。線上拜拜的核心價值之一即是環保，但目前平台缺乏量化呈現此效益的機制。透過具體數據與視覺化展示，能讓使用者「有感」地理解自己的環保貢獻，進而提升平台黏著度與使用動機。

## 範圍

### 包含

- 單次儀式環保效益計算（紙張減少量、碳排放減少量）
- 已登入使用者的累積環保成效統計
- 訪客使用者的本機環保紀錄
- 環保數據視覺化（等效換算比喻、趨勢圖表）
- 環保里程碑達成激勵機制

### 不包含

- 社群分享功能（未來迭代）
- 與外部環保組織的合作認證
- 環保排行榜

## 影響評估

- **使用者體驗**：儀式完成後增加環保回饋環節，強化正向感受
- **技術面**：需新增環保計算引擎、資料儲存結構、圖表元件
- **效能面**：圖表採用 Apache ECharts + tree-shaking，確保行動裝置效能
