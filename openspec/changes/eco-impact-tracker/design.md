# 設計：環保影響追蹤器

## 架構概覽

環保影響追蹤器由四個主要模組組成：計算引擎、資料層、視覺化元件、里程碑系統。

```
┌─────────────────────────────────────────────┐
│              前端展示層                       │
│  ┌──────────┐ ┌───────────┐ ┌─────────────┐ │
│  │ 儀式結果  │ │ 成效總覽  │ │ 里程碑通知  │ │
│  │ 環保卡片  │ │ 趨勢圖表  │ │ 動畫元件    │ │
│  └────┬─────┘ └─────┬─────┘ └──────┬──────┘ │
│       │             │              │         │
│  ┌────┴─────────────┴──────────────┴──────┐  │
│  │         等效換算展示元件                │  │
│  └────────────────┬───────────────────────┘  │
└───────────────────┼──────────────────────────┘
                    │
┌───────────────────┼──────────────────────────┐
│              計算引擎                         │
│  ┌────────────────┴───────────────────────┐  │
│  │  EcoImpactCalculator                   │  │
│  │  - 金紙重量對照表                       │  │
│  │  - 紙紮物品重量對照表                   │  │

│  │  - 等效換算公式                         │  │

└───────────────────┼──────────────────────────┘
                    │

│  │  訪客：localStorage 本機儲存           │  │
│  └────────────────────────────────────────┘  │
└──────────────────────────────────────────────┘
```

## 模組設計

### 1. 環保效益計算引擎

**職責**：根據儀式中使用的金紙種類與紙紮物品，計算對應的環保效益。

**金紙重量對照資料**：

| 金紙種類 | 單份估計重量（公克） |
|---------|-------------------|
| 大百壽金 | 150 |
| 壽金     | 120 |
| 刈金     | 80  |
| 福金     | 60  |
| 銀紙     | 100 |

**紙紮物品**：依物品類別提供預設重量估算值。

**碳排放計算**：
- 每公克紙張燃燒約產生 1.8 公克 CO₂
- 計算公式：`碳排放減少量 = 紙張總重量 × 1.8`

**等效換算**：
- 1 棵樹每年吸收約 12,000 公克 CO₂
- `拯救樹木數 = 累積碳排放減少量 / 12000`

### 2. 資料層

**已登入使用者**：
- 每次儀式完成後，將環保數據寫入伺服器端資料庫
- 儲存欄位：儀式 ID、日期、紙張減少量、碳排放減少量、使用的金紙與紙紮明細
- 提供累積查詢與按月/年分組查詢 API

**訪客使用者**：
- 使用 `localStorage` 儲存環保紀錄
- 資料結構與已登入使用者相同，但標示為「訪客紀錄」
- 訪客登入後可選擇將本機紀錄合併至帳戶

### 3. 視覺化元件

**環保卡片**：
- 儀式完成後即時顯示
- 呈現本次減少的紙張量與碳排放量
- 包含等效換算比喻文字

**趨勢圖表**：
- 使用 Apache ECharts（Canvas-based）
- 透過 tree-shaking 只引入所需模組（BarChart、LineChart、GridComponent、TooltipComponent）
- 支援按月/年切換檢視
- 響應式設計，適配行動裝置螢幕

**等效換算展示**：
- 將累積數據換算為日常生活易懂的比喻
- 範例：「相當於拯救 N 棵樹」、「減少 N 公斤碳排放」

### 4. 里程碑系統

**里程碑定義**：

| 里程碑名稱       | 條件                    |
|-----------------|------------------------|
| 環保新手         | 完成首次線上拜拜          |
| 綠色守護者       | 累積減少 1 公斤碳排放     |
| 森林之友         | 相當於拯救 1 棵樹        |
| 環保達人         | 完成 50 次線上拜拜       |
| 地球守護者       | 相當於拯救 10 棵樹       |

**里程碑達成通知**：
- 偵測到達成新里程碑時，觸發祝賀動畫
- 動畫使用 CSS animation，不額外引入動畫庫
- 里程碑紀錄存於使用者資料中

## 技術選型

| 項目 | 選擇 | 理由 |
|------|------|------|
| 圖表庫 | Apache ECharts | Canvas-based，行動裝置效能佳，支援 tree-shaking |
| 訪客儲存 | localStorage | 無需登入即可保留紀錄 |
| 動畫 | CSS Animation | 輕量，無需額外依賴 |

## 效能考量

- ECharts 透過 tree-shaking 僅打包所需模組，預估增加約 100-150 KB（gzipped）
- 計算引擎為純函式，可在前端直接執行，無需額外 API 呼叫
- 趨勢圖表資料按需載入，避免一次性載入全部歷史資料
