## Why

線上拜拜儀式中，點香與焚香是核心環節。使用者需要透過點香來表達虔誠，而香的燃燒過程也象徵著祈願的傳達。目前系統尚未提供點香與焚香的視覺模擬，導致儀式體驗不完整。此外，焚香通常需要一段時間，使用者可能在過程中離開頁面，因此需要支援背景計時與推播通知，確保使用者不會錯過焚香完成的時刻。

## What Changes

- 新增點香動畫，使用者點擊「點香」按鈕後播放火焰點燃與煙霧生成效果
- 新增焚香煙霧持續飄散效果，香的長度隨時間遞減
- 新增焚香背景計時功能，透過 Service Worker 在背景持續運作，計時結束時間戳記錄至 IndexedDB
- 新增焚香完成推播通知，透過 Web Notification API 提醒使用者「香已燃畢」
- 新增回到頁面自動狀態恢復，根據 IndexedDB 時間戳計算剩餘時間並恢復動畫進度

## Capabilities

### New Capabilities
- `incense-simulation`: 點香與焚香視覺模擬，包含動畫效果、背景計時、推播通知與狀態自動恢復

### Modified Capabilities
_None_

## Impact

- **前端動畫**：新增點香火焰動畫、煙霧粒子效果、香長度遞減動畫
- **Service Worker**：新增背景計時邏輯，支援頁面關閉後繼續計時
- **本機儲存**：使用 IndexedDB 持久化焚香計時結束時間戳
- **推播通知**：整合 Web Notification API，處理通知權限請求與不支援平台的降級方案
- **狀態管理**：新增焚香狀態（進行中、已完成），支援頁面可見性變更時自動恢復
